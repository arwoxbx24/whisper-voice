# Whisper Voice — Руководство пользователя

## Что это такое

Whisper Voice — программа для голосового ввода текста. Вы нажимаете горячую клавишу, говорите в микрофон, и текст появляется там, где находится курсор — в браузере, мессенджере, Word, редакторе кода, в любом приложении.

Распознавание речи выполняется через облачный сервис OpenAI Whisper — тот же, что используется в ChatGPT. Качество распознавания русского языка очень высокое.

---

## Системные требования

- Windows 10 или Windows 11
- Микрофон (встроенный в ноутбук или внешний)
- Интернет (для облачного распознавания)
- Аккаунт на platform.openai.com (для получения API ключа)

---

## Установка

1. Скачайте `WhisperVoice.exe` из раздела [Actions](../../actions) — выберите последнюю успешную сборку, скачайте артефакты
2. Запустите `WhisperVoice.exe`
3. Если Windows показывает предупреждение "Неизвестный издатель" — нажмите "Подробнее" → "Выполнить в любом случае"
4. Откроется мастер настройки — следуйте инструкциям на экране

---

## Первый запуск — Мастер настройки

При первом запуске автоматически открывается мастер настройки из 4 шагов:

**Шаг 1 — Добро пожаловать**
Ознакомьтесь с описанием приложения. Нажмите "Далее".

**Шаг 2 — API ключ**
Введите ваш OpenAI API ключ. Нажмите "Проверить ключ" чтобы убедиться, что ключ рабочий.
Если ключа ещё нет — можно пропустить этот шаг и ввести позже через "Настройки" в трее.

**Шаг 3 — Горячая клавиша**
Нажмите "Задать клавишу", затем зажмите нужную комбинацию на клавиатуре.
По умолчанию: `Ctrl + Shift + Space`.
Также выберите режим записи:
- **Переключение** — нажал один раз → запись начинается, нажал снова → останавливается
- **Удержание** — держишь клавишу → запись идёт, отпустил → останавливается

**Шаг 4 — Готово**
Проверьте итоговые настройки и нажмите "Начать работу". Программа свернётся в системный трей (область уведомлений рядом с часами).

## Как получить API ключ OpenAI

1. Перейдите на [platform.openai.com](https://platform.openai.com)
2. Зарегистрируйтесь или войдите в аккаунт
3. В левом меню нажмите **API Keys**
4. Нажмите кнопку **Create new secret key**
5. Введите название ключа (например, "Whisper Voice") и нажмите **Create secret key**
6. Скопируйте ключ — он начинается с `sk-`. Сохраните его, так как он показывается только один раз
7. Вставьте ключ в поле мастера настройки Whisper Voice

> Новые аккаунты OpenAI получают бесплатный кредит на $5 для тестирования.

## Использование

1. Программа работает в фоне — иконка видна в системном трее (правый нижний угол экрана)
2. Переключитесь в приложение, куда хотите вставить текст (браузер, Word, Telegram и т.д.)
3. Нажмите горячую клавишу `Ctrl + Shift + Space` — запись начнётся
4. Говорите в микрофон
5. Нажмите горячую клавишу снова (в режиме "Переключение") или отпустите (в режиме "Удержание")
6. Через секунду-две текст автоматически появится в активном приложении

---

## Режимы записи

### Режим "Переключение" (Toggle) — рекомендуется

Один раз нажали — запись началась. Говорите сколько угодно. Нажали снова — запись остановилась, текст вставился.

Удобно для длинных фраз и диктовки.

### Режим "Удержание" (Hold)

Зажали клавишу — запись идёт. Отпустили — запись остановилась, текст вставился.

Удобно для коротких команд, когда нужно быстро надиктовать одно слово или фразу.

---

## Провайдеры распознавания речи

По умолчанию используется OpenAI Whisper. В файле настроек можно включить дополнительные провайдеры:

| Провайдер | Стоимость | Работа без интернета |
|-----------|-----------|---------------------|
| OpenAI Whisper | ~$0.006/мин (~0.5 руб) | Нет |
| Deepgram Nova-3 | ~$0.0043/мин (~0.4 руб) | Нет |
| Local (faster-whisper) | Бесплатно | Да |

При указании нескольких провайдеров — если один недоступен, программа автоматически переключается на следующий.

## Настройки

**Через мастер настройки**: правый клик на иконке в трее → "Настройки"

**Через файл конфигурации**: `%USERPROFILE%\.whisper-voice\config.json`

Основные параметры:

```json
{
  "api_key": "sk-...",
  "hotkey": "<ctrl>+<shift>+space",
  "hotkey_mode": "toggle",
  "language": "ru",
  "stt_providers": ["openai"]
}
```

Для изменения горячей клавиши через файл — используйте формат `<ctrl>+<shift>+space`, где модификаторы берутся в угловые скобки.

## Часто задаваемые вопросы (FAQ)

### Где взять API ключ OpenAI?
Перейдите на [platform.openai.com/api-keys](https://platform.openai.com/api-keys), войдите в аккаунт, нажмите "Create new secret key". Ключ начинается с `sk-`. Новые аккаунты получают бесплатный кредит $5.

### Приложение не запускается / показывает ошибку при запуске
- Попробуйте запустить от имени администратора (правый клик → "Запуск от имени администратора")
- Проверьте, не блокирует ли антивирус — добавьте исключение для `WhisperVoice.exe`
- Убедитесь, что Windows 10 или 11 (более ранние версии не поддерживаются)

### Текст не вставляется в приложение
- Убедитесь, что курсор стоит в текстовом поле нужного приложения перед нажатием горячей клавиши
- Попробуйте нажать `Ctrl+V` вручную после распознавания
- Некоторые приложения с особой защитой (например, системные диалоги) могут блокировать автоматическую вставку

### Горячая клавиша не срабатывает
- Убедитесь, что программа запущена (иконка в трее)
- Проверьте, не занята ли клавиша другой программой
- Смените горячую клавишу через Настройки

### Плохое качество распознавания речи
- Говорите чётче, в нормальном темпе
- Проверьте, что микрофон работает (Настройки Windows → Звук → Ввод)
- Попробуйте говорить ближе к микрофону
- Уменьшите фоновый шум
- Попробуйте другой провайдер (Deepgram иногда даёт лучшее качество)

### Сколько стоит использование?
- OpenAI Whisper: ~$0.006 за минуту (~0.5 руб/мин)
- Deepgram Nova-3: ~$0.0043 за минуту (~0.4 руб/мин)
- Local (faster-whisper): бесплатно, но требует мощного компьютера и работает медленнее

При обычном использовании (10-30 минут диктовки в день) стоимость составляет несколько рублей в месяц.

### Как изменить горячую клавишу?
Правый клик на иконке в трее → "Настройки" → откроется мастер настройки, перейдите к шагу "Горячая клавиша".

### Работает ли без интернета?
Только с провайдером `local` (faster-whisper). Для его активации откройте файл `%USERPROFILE%\.whisper-voice\config.json` и измените:
```json
"stt_providers": ["local"]
```
Учтите, что первый запуск потребует загрузки модели (~150 МБ).

### Как посмотреть что происходит / найти ошибку?
Откройте файл `%USERPROFILE%\.whisper-voice\whisper-voice.log` в любом текстовом редакторе.
